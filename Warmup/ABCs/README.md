# ABCs

## Problem Description

Given three integers $A$, $B$, and $C$, determine their sum.

Your task is to implement the function ```getSum(A, B, C)``` which returns the sum $A + B + C$.

## Constraints

$1 \leq A,B,C \leq 100$

## Approach

### High-Level Solution

1. Simply return $A$ + $B$ + $C$.
